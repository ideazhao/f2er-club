# itbus.club 
为广大同仁提供一个好的工具、资源平台

受众：前端技术学习者、爱好者、从业者

宗旨：做“最懂你、最贴心”的前端资源网

##主要内容

涉及一些优秀攻城狮的优秀博客、社区，实用的手册、工具，框架/库，以及在线教程和书籍推荐。

为大家解决一些基本的，常见的问题和需求。

1、我们没有必要每次要找某个网站都去百度或者google，也没有必要把我们的网址收藏夹弄的满满的，要找的时候很不方便。

2、是的，已经有类似的站点，但是我觉得还不够，从广度，细度以及跟专业需求匹配度上，都还有差距，与其疲于去记录和保存一个个零散的站点和文章，那何不把它们汇聚到这里来？
